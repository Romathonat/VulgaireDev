Proposer un article

Pour proposer un article, il vous suffit de suivre le lien suivant : Proposer un Article.
Celui-ci vous amènera sur mon repo Github, où sont présent les articles. Il vous suffit de fork le repo, d'ajouter un nouveau dossier, et de faire une pull request. L'interface markdown de Github est très bien, l'intégration des articles via des pull request est facile, github est un outil très populaire chez les développeurs, et avec ce système vous pouvez rédiger un début d'article, le sauvegarder, puis le reprendre et le soumettre quand vous le souhaitez.

En détails

  1. Suivez ce lien
  2. Cliquez sur "Fork" en haut à droite de la page. Ceci va créer une copie du dossier dans votre espace Github.
  3. Rendez vous dans votre espace Github, cliquez sur votre dossier "vulgaireDevEntries".
  4. Cliquez sur "Create new file". Choisissez le nom de votre fichier, et faites le finir par ".md" (pour créer un dossier jetez un oeil ici).
  5. Pour sauvegarder votre avancement, donnez un nom au travail que vous avez effectué (par exemple "debut de redaction"), puis cliquez sur commit. Vous pourrez ainsi revenir finir votre article quand vous le souhaiterez.
  6. Une fois fini, pour faire une pull request (càd une demande d'intégration de votre travail dans vulgaireDevEntries qui est chez moi), cliquez sur "New pull request" (en haut à gauche), puis "Create Pull Request".
  7. Voilà, plus qu'à attendre le validation !
